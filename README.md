# tgfs

Use telegram as file storage

## Installation

```bash
$ yarn install && yarn build
$ alias tgfs="yarn cmd"
```

## usage

- ls

  ```bash
  $ tgfs ls /
  ```

- mkdir

  ```bash
  $ tgfs mkdir /documents
  ```

- cp

  ```bash
  $ tgfs cp ~/some-file /
  ```

- rm

  ```bash
  $ tgfs rm /some-file
  ```
